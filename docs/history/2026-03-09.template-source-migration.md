# Migration History: Template Source Migration

Date: 2026-03-09  
Repository: `flatrick/modeldev-toolkit`

## Purpose

This document records the completed template-source migration that moved tracked
tool adapter source out of repo-root runtime-style directories such as
`.claude/`, `.cursor/`, `.codex/`, and `.opencode/`.

The goal was to make canonical source explicit, keep runtime/install outputs out
of tracked source ownership, and reduce accidental drift between shipped
templates and local tool state.

## Final Outcome

The migration completed with these accepted source locations:

- `claude-template/`
- `cursor-template/`
- `codex-template/`
- `opencode-template/`

The corresponding runtime/install dirs are now treated as local state or
rendered output only:

- `.claude/`
- `.cursor/`
- `.codex/`
- `.opencode/`

## Final Accepted Layout

- `claude-template/hooks.json` is the canonical Claude hook config source.
- `hooks/hooks.json` remains the synced Claude-facing mirror and is not the
  source of truth.
- `cursor-template/` is the tracked Cursor adapter source.
- `codex-template/` is the tracked Codex adapter source.
- `opencode-template/` is the tracked OpenCode adapter source.
- `CLAUDE.md` and `.claude-plugin/` intentionally remain at repo root as
  Claude-native project guidance and plugin metadata, not as migration leftovers.
- Shared MDT assets such as schemas, commands, agents, skills, and scripts stay
  at repo root.
